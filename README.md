A classic Snake Game built using HTML, CSS, and JavaScript.
The player controls a snake that grows longer as it eats food while avoiding walls and itself.                                      Start Game button
* Restart Game button
* Score tracking system
* Game Over screen
* Smooth movement controls (keyboard arrows / touch optional)                              Built With

* HTML – structure
* CSS – styling and layout
* JavaScript – game logic.                    How to Run the Game

1. Download or clone this repository
2. Open the folder
3. Click on index.html
4. The game will open in your browser.        Controls

* ⬆️ Arrow Up → Move up
* ⬇️ Arrow Down → Move down
* ⬅️ Arrow Left → Move left
* ➡️ Arrow Right → Move right
